## Global Rules

### Core Mandate: Topic & Complexity

**Topic Integrity**
Strictly adhere to the physical system and variables defined in the seed. However, you are **authorized and expected** to introduce theoretical hurdles to elevate the problem to an **expert research level** reasoning. The problem should be challenging even for an advanced theoretical physicist, requiring novel synthesis of concepts rather than textbook pattern-matching.

**Vectors of Difficulty**
To achieve this standard, you must leverage one or more of the following strategies. Do not use all of them at once; choose the strategy that best fits the specific physics topic to create a coherent, rigorous challenge.

1.  **Reasoning Depth:** The path to the solution must not be obvious. It should require **multi-stage logical inference** where the student must first determine *which* framework applies, then construct the specific operators or invariants, and finally execute the calculation. Ambiguity in the setup that requires physical argumentation to resolve is encouraged.
2.  **Mathematical Sophistication:** Require the use of advanced formalisms (e.g., contour integration, differential geometry, group theory weights, heat kernel methods, asymptotic matching) rather than simple algebraic substitution.
3.  **Conceptual Insight:** Construct the problem such that the solution requires identifying a subtle physical feature—such as a hidden symmetry, a specific vacuum structure, an anomaly, or a consistency constraint (e.g., unitarity bounds, gauge invariance)—that is not explicitly stated but is essential to the result.
4.  **Cross-Domain Injection:** Where appropriate, embed the core topic in a more complex context. Some possible examples:
    * *Geometry:* Place the system in a curved background, on a lattice, or in a finite volume.
    * *Thermodynamics:* Introduce a chemical potential or finite temperature.
    * *Topology:* Introduce boundary conditions, defects (monopoles/vortices), or compact dimensions.
    * *etc...:* You should consider more Cross-Domain Injection that are appropriate for the topic. 

**Constraint:** The resulting problem must remain physically solvable and consistent. The added complexity must be *necessary* to the task, not just flavor text.

**Difficulty** - The problem should be **extremely challenging**. It should be formatted as a high-level task where the student must navigate the technical complexity from first principles.

- **Mathematics:** Require the use of advanced graduate-level mathematical formalisms. The solution must move beyond simple algebraic substitution or power-law scaling. The path to the solution should require navigating structures like branch cuts, operator determinants, non-commuting variables, or differential geometry constraints.

- **Conceptual Depth (No Shortcuts):** Do not provide the mathematical "rules" of the system (e.g., the specific form of a tensor or the projector). The student must:
    - **Derive/Construct** the required mathematical objects (tensors, operators, or invariants) from the symmetries, conservation laws, or first principles.
    - Justify the use of specific **regularization or renormalization schemes** if the derivation requires handling singularities.

- **Complexity:** Every task must require at least **three distinct logical or algebraic leaps** to reach the final coefficient:
    1. **Formal Construction:** Defining the mathematical framework from the physical principles provided.
    2. **Analytical Rigor:** Performing a non-trivial algebraic operation (e.g., a complex tensor reduction or a $d$-dimensional trace).
    3. **Precision Extraction:** Computing the final numerical or algebraic constant through a rigorous evaluation (e.g., solving a differential equation or evaluating a residue).

- **Anti-Guessing Constraint:** The final result must not be obtainable via dimensional analysis or "common sense" alone. It should be sensitive to **relative signs, factors of $i$, phase factors**, or specific group theory invariants (e.g., $C_A, C_F, T_R$) and combinatorial factors that require full derivation to determine.

- **Tone and Style** – Use a strict, high-level examination tone (e.g., PhD Qualifying Exam). Be terse and direct. **Do not** be "helpful" or "pedagogical".
    1. **Banned Phrases:** Avoid "Recall that...", "It is well known that...", "Note that...", or "As discussed in...".
    2. **Requirement:** State only the necessary physical system specifications (Lagrangian/Symmetry) and the Task. Assume the student is an expert who does not need definitions of standard terms (e.g., do not define "Hermitian" or "renormalizable").
**Evaluable outputs** –

Each task must produce one or more outputs that can be checked automatically.

Outputs can be:

- **Numerical** → real or complex scalars, ratios, constants, signs, exponents, or boolean values (True/False, 1/0).
- **Categorical** → discrete labels chosen from a small known set (e.g., ["scalar","pseudoscalar"], ["U(1)","SU(2)","SU(3)"], etc.).

Do not require symbolic expression comparison.

**Cancellation / reduction rule** –
If any analytic pieces cancel or combine, hide that step internally and ask only for the final surviving coefficients or categories.


## Guidelines: Task Types and Structure

The task in section A) Problem must belong to one of the following verifiable categories.
task may combine several reasoning steps (e.g., derivation → numeric coefficient), but the final output must always be checkable either numerically or categorically.

### 1. Direct Calculation task

Students compute a quantity that can be directly expressed as a number or numerical function of given parameters.

**Examples:**

- Evaluate a regulated integral to find a finite constant $C_1(d,n)$.
- Compute the scaling exponent $\alpha(d,n)$ from dimensional analysis.
- Determine whether a matrix element vanishes (boolean output True/False).

### 2. Derivation or Identification task

Students perform a conceptual or algebraic derivation, but the final deliverable is a categorical label or numerical value.

**Examples:**

- Identify the Lorentz or parity class of a bilinear (categorical).
- Determine if a given operator is renormalizable (categorical or boolean).
- Derive and report the numerical coefficient of a divergence.

### 3. Hidden-Coefficient Derivation task

The most common "derivation-by-numbers" structure.

The prompt hides known analytic results and asks for specific constants, coefficients, or exponents that appear in the final expression.

Students must perform any required derivation internally but only report these numeric values.

**Examples:**

- Find constants $C_R$, $C_L$ in a Hamiltonian.
- Compute coefficients $c_a$, $c_{a5}$ in a current decomposition.
- Determine $K$ such that $M = K C_W$.
- Find $C(N)$ such that $\Omega = C(N) \lambda^2$


### 4. Ratio / Comparison task

Students compare two physically related quantities and output a numeric or boolean ratio or comparison result.

**Examples:**

- Compute $R = I_2 / I_1$.
- Check whether $R > 0$ (boolean).
- Determine relative sign between two amplitudes.

### 5. Categorical Classification task

Students identify the correct discrete label from a finite set (string output).

**Examples:**

- "scalar", "pseudoscalar", "vector", "axial-vector".
- "bosonic" vs "fermionic".
- Gauge group identification: "U(1)", "SU(2)", "SU(3)".

### 6. Logical or Consistency-Check task

Students verify whether a derived quantity satisfies a condition, outputting True/False or 1/0.

**Examples:**

- Check if the Hamiltonian is Hermitian.
- Check for gauge invariance or conservation of a current.

Example 1: Single-task problem (Harder)

\section{Problem}
Consider an $SU(3)$ gauge theory coupled to $N_f$ flavors of Dirac fermions in the fundamental representation. 

\textbf{Task:}
Derive the one-loop beta function $\beta(g)$ from the relevant Feynman diagrams (vertex and propagator corrections). Determine the maximum integer number of fermion flavors $N_f^{max}$ such that the theory remains asymptotically free.

\textbf{Conventions:} The beta function is defined as $\beta(g) = \mu \frac{dg}{d\mu}$. Asymptotic freedom corresponds to $\beta(g) < 0$ for $g>0$.

\section{Problem Description}
This problem requires the student to recall or derive the non-Abelian beta function coefficients, specifically the contribution of the gauge field (gluons) and the fermion matter content, to determine the loss of asymptotic freedom.

\section{Answer Requirements}
```python
def get_max_flavors(N_colors: int) -> int:
    """
    Derives the asymptotic freedom bound for SU(N) with fundamental fermions.
    
    Args:
        N_colors: The number of colors N in SU(N).
        
    Returns:
        int: The maximum number of flavors N_f allowed.
    """
    raise NotImplementedError
```

\section{Solution}
The one-loop beta function is given by $\beta(g) = -\frac{g^3}{(4\pi)^2} b_0$.

The coefficient $b_0$ is derived from the vacuum polarization and vertex corrections:
\begin{equation}
b_0 = \frac{11}{3} C_2(G) - \frac{4}{3} N_f T(R)
\end{equation}

For $SU(N)$, the adjoint Casimir is $C_2(G) = N$ and the fundamental index is $T(R) = 1/2$.

Substituting these values:
\begin{equation}
b_0 = \frac{11}{3}N - \frac{4}{3}N_f \left(\frac{1}{2}\right) = \frac{11}{3}N - \frac{2}{3}N_f
\end{equation}

For asymptotic freedom, we require $\beta(g) < 0$, which implies $b_0 > 0$:
\begin{equation}
\frac{11}{3}N - \frac{2}{3}N_f > 0 \implies N_f < \frac{11}{2}N
\end{equation}

For $SU(3)$, $N=3$, so $N_f < 16.5$. The maximum integer is 16.

\section{Answer}
The maximum integer number of flavors is 16.

\section{Code}
```python
import math

def get_max_flavors(N_colors: int) -> int:
    # The limit is derived as N_f < (11 * N) / 2
    limit = (11 * N_colors) / 2
    # We need the largest integer strictly less than the limit if limit is integer, 
    # or floor if it's float.
    if limit.is_integer():
        return int(limit) - 1
    else:
        return math.floor(limit)
```

Example 2: Multi-task problem (Harder)

\section{Problem}
In a scalar field theory in $d$-dimensional Euclidean space, consider the one-loop self-energy correction $\Sigma(p)$. 

\textbf{Tasks:}
Analytically continue the dimension to $d = 4 - \epsilon$. Perform the loop integration using Feynman parameters and extract the coefficient of the $1/\epsilon$ divergence for the massive bubble diagram.

Based on the mass dimension of the coupling constant $\lambda$ in a $\phi^n$ interaction term, determine the critical spacetime dimension $d_c$ where a $\phi^3$ interaction becomes dimensionless (renormalizable).

\textbf{Conventions:} The loop integral measure is $\mu^\epsilon \int \frac{d^d k}{(2\pi)^d}$. The divergent part is defined as the coefficient of the pole term $1/\epsilon$.

\section{Problem Description}
The student must perform a full dimensionally regulated integration to isolate the pole, rather than just identifying it from a table. They must also apply dimensional analysis to field operators.

\section{Answer Requirements}
```python
def loop_and_dim_analysis(n_interaction: int) -> tuple[float, float]:
    """
    Args:
        n_interaction: The power n in the phi^n interaction term.

    Returns:
        tuple[float, float]: 
            - pole_coeff: The numeric coefficient of the 1/epsilon term in the 1-loop integral.
            - critical_dim: The spacetime dimension where phi^n is renormalizable.
    """
    raise NotImplementedError
```

\section{Solution}
Task 1:
The standard one-loop integral is $I = \int \frac{d^d k}{(2\pi)^d} \frac{1}{(k^2+m^2)^2}$.

Using Gamma function identities:
\begin{equation}
I = \frac{\Gamma(2-d/2)}{(4\pi)^{d/2} (m^2)^{2-d/2}}
\end{equation}

Expanding near $d=4-\epsilon$, we use $\Gamma(\epsilon/2) \sim 2/\epsilon$.

The factor $(4\pi)^{-(2-\epsilon/2)} \approx \frac{1}{(4\pi)^2}(1 + \dots)$.

The pole term comes from $\Gamma(\epsilon/2)$, giving a coefficient of $\frac{2}{(4\pi)^2 \epsilon}$.

The problem asks for the coefficient of the pole relative to the loop factor. If we normalize by $1/(4\pi)^2$, the coefficient is 2.

Task 2:
The action $S = \int d^d x \mathcal{L}$ is dimensionless. $[\mathcal{L}] = d$.

The kinetic term $(\partial \phi)^2$ implies $2 + 2[\phi] = d \implies [\phi] = (d-2)/2$.

For an interaction $\lambda \phi^n$, we need $[\lambda \phi^n] = d$.

For the theory to be strictly renormalizable, $[\lambda] = 0$.

$0 + n[\phi] = d \implies n(d-2)/2 = d$.

$n d - 2n = 2d \implies d(n-2) = 2n \implies d_c = \frac{2n}{n-2}$.

For $n=3$ ($\phi^3$ theory): $d_c = \frac{6}{3-2} = 6$.

\section{Answer}
The pole coefficient is 2.0. The critical dimension for $\phi^3$ is 6.0.

\section{Code}
```python
def loop_and_dim_analysis(n_interaction: int) -> tuple[float, float]:
    # Task 1: The pole of Gamma(2 - d/2) at d=4 is 2/epsilon.
    pole_coeff = 2.0
    
    # Task 2: d_c = 2n / (n - 2)
    if n_interaction == 2:
        critical_dim = float('inf') # Free theory
    else:
        critical_dim = (2 * n_interaction) / (n_interaction - 2)
        
    return (pole_coeff, critical_dim)
```
